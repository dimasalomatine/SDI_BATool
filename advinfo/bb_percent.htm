Bollinger Bands %

"Bollinger Bands %" has two bands plotted around a moving average using the value of standard deviation. An indicator derived from "Bollinger Bands" that we call "Bollinger Bands %" can be of great help, using the same formula that George Lane used for stochastics. The indicator "Bollinger Bands %" tells us where we are within the bands.
